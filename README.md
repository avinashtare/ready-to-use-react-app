# React + Vite + tailwindcss

Create a react app fast with additinal components.

```shell
yarn clone https://github.com/avinashtare/ready-to-use-react-app.git
```


```shell
cd ready-to-use-react-app
```

```shell
yarn
```

```shell
yarn run dev
```
